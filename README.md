# ✈️ Калькулятор расходов для путешествий

Учебный проект по дисциплине «Технологии разработки ПО». Веб-приложение на Flask для контроля бюджета и фиксации расходов в поездках.

## ✨ Функционал
- Установка и динамическое изменение общего лимита бюджета.
- Добавление расходов по категориям (Еда, Жилье, Transport, Развлечения).
- Удаление записей и автоматический пересчёт остатка.
- Фильтрация истории трат по категориям.
- Визуальный индикатор (Progress Bar) заполнения бюджета.

## 🛠️ Стек технологий
- **Backend:** Python 3, Flask
- **Database:** SQLite 3
- **Frontend:** HTML5, Bootstrap 5
