Руководство пользователя

Приложение "Калькулятор расходов для путешествий" позволяет контролировать бюджет в поездках.

Как пользоваться:
1. **Изменение бюджета:** В левом верхнем углу в поле "Новый бюджет" введите общую сумму, выделенную на поездку, и нажмите кнопку "Изм.".
2. **Добавление расходов:** В форме "Добавить расход" выберите категорию (Еда, Жилье, Транспорт, Развлечения), введите сумму и описание траты. Нажмите "Внести трату".
3. **Фильтрация:** В правом верхнем углу таблицы "История расходов" выберите нужную категорию из выпадающего списка, чтобы отфильтровать записи.
4. **Удаление:** Для удаления ошибочно внесенной траты нажмите красную кнопку "❌ Удалить" напротив соответствующей записи.
