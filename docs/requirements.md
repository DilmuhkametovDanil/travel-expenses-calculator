#Техническое задание: Калькулятор расходов для путешествий (requirements.md)

1. Проблема
При планировании поездок путешественники часто выходят за рамки заложенного бюджета, так как мелкие расходы (еда, сувениры, транспорт) трудно удержать в голове. Приложение помогает наглядно фиксировать каждую трату в режиме реального времени и видеть, сколько денег осталось до критической отметки.

2. Целевая аудитория
 Студенты, планирующие бюджетные поездки и каникулы.
 Путешественники, желающие контролировать свои финансы в дороге.

3. Функциональные требования
1. Создание поездки с указанием общего бюджета.
2. Добавление трат по категориям (Еда, Жилье, Транспорт, Развлечения) с указанием суммы.
3. Просмотр списка всех трат и автоматический подсчет общей суммы расходов.
4. Отображение остатка бюджета и визуальный индикатор (полоска прогресса) израсходованных средств.
5. Фильтрация или сортировка трат по категориям или величине суммы.

4. Архитектура приложения
* Клиент (Браузер): HTML5, CSS3 (фреймворк Bootstrap для адаптивного интерфейса и прогресс-бара).
* Веб-сервер (Flask): обработка маршрутов, расчет остатка бюджета, передача данных в шаблоны.
* Хранилище (SQLite): база данных для хранения лимита бюджета и списка внесенных расходов.
